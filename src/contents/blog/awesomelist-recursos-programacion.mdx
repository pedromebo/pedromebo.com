> ## Introducción
En el post de hoy vamos a hablar de un conjunto de repositorios de GitHub lleno de recursos, proyectos y libros que nos pueden ayudar muchísimo de cara a aprender una nueva tecnología o a profundizar en una de ellas.

Estos recursos se denominan "Awesome" y se trata de repositorios creados por usuarios que listan contenido de gran valor para investigaciones, proyectos, herramientas, libros, tutoriales o guías.

La mayoría de ellos se tratan de enlaces a otros repositorios, portfolios, videos o libros.

Gran parte de las listas "awesome" están agrupadas en este repositorio donde podrás buscar y filtrar lo que necesites.

[*Ir a la página de GitHub: Awesome list*](https://github.com/sindresorhus/awesome)

> ## ¿Cómo buscar contenido dentro del proyecto Awesome?
Este proyecto del usuario [*sindresorhus*](https://github.com/sindresorhus/) tiene más de 143k de estrellas y casi 19k forks.

Por ejemplo, pongamos el caso de que queremos aprender a hacer Machine Learning en Python. Primero buscamos Python y vemos un conjunto de subgrupos.

![Python Subgrupos](https://i.gyazo.com/2bb4ca1364962901dd0b26c20b4edfef.png)

[![Awesome Data Science](https://i.gyazo.com/9a3ac425d34ae362cbe4be64bdf80806.png)](https://github.com/krzjoa/awesome-python-data-science)

Esto nos redirigirá a un nuevo proyecto Awesome pero esta vez solo para [Data Science](https://github.com/krzjoa/awesome-python-data-science).
